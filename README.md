# Qv2ray deps

This repo contains [GitHub Action Scripts](https://github.com/Qxray-app/Qv2ray-deps/tree/master/.github/workflows) which can build Qv2ray dependencies automatically

## Contents:
- gRPC (protobuf)
- cURL
- OpenSSL

## Supported Platforms:
- Android
  - ARM
  - ARM64
  - x86
  - x86_64
 
- Windows
  - x86
  - x86_64
